# Echo Dashboard

Auto-deployed at: https://echo-dashboard.netlify.app

## How to Update

1. Go to https://github.com and sign in (or create account)
2. Create a new repository named "echo-dashboard"
3. Upload `echo-dashboard.html` to the repo
4. Go to https://app.netlify.com
5. "Add new site" → "Import an existing project"
6. Select GitHub → select "echo-dashboard" repo
7. Done! Auto-deploys on every file change

## Auto-Deploy Setup

Once connected:
- Upload new `echo-dashboard.html` to GitHub
- Netlify auto-deploys in ~30 seconds
- Refresh your dashboard URL
